# Persistência de Dados com Python:

O trabalho tem como objetivo praticar os seguintes itens requisitados:

- Oordenação de listas em Python;
- Bubble Sort;
- Selection Sort;
- Leitura de arquivos `.txt`;
- Escrita de arquivos `.txt`.

# Arquivos do projeto:

- array.sort.py
Utiliza o método `sort()` do Python para ordenar listas em ordem crescente e decrescente.

- bubble.sort.py
Implementa o algoritmo Bubble Sort.

- selection.sort.py
Implementa o algoritmo Selection Sort.

- ler.txt.py
Faz a leitura do arquivo `loremipsum.txt`.

- escrever.txt.py
Cria um arquivo `.txt` e escreve frases nele.

- kdd.py (arquivo principal do projeto)  
Lê um arquivo de texto, separa as palavras, ordena utilizando diferentes métodos e compara o tempo de execução de cada um.

# Comando para executar

```bash
python kdd.py
